Opis szczegółowy

Klasa wyświetlająca okno programu.

Składa się tylko z dwóch pól:

  • po lewej kolorowa reprezentacja otrzymanych obiektow ImageWidget
  • po prawej panel do nawiązania połączenia BConnectionWidget
