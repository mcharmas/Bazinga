Opis szczegółowy

Klasa ta jest menagerem uzytkownikow, grup i sterownikow. Gdy uzytkownik jest podlaczany ta klasa zarzadza do jakiej grupy idzie.
Klasa dba tez o to aby puste grupy byly usuwane.
